Comment fonctionne NetGuard Pro
🔄
Architecture en 3 couches
NetGuard Pro fonctionne avec un backend Python qui capture les paquets réseau, un moteur de détection qui analyse chaque paquet en temps réel, et un tableau de bord web connecté via WebSocket pour tout visualiser.
1
Capture — Scapy intercepte tous les paquets réseau qui transitent par ton interface (Wi-Fi ou Ethernet) grâce au driver Npcap sur Windows.
2
Analyse — Chaque paquet est analysé en profondeur : IP, port, protocole, flags TCP, DPI (injection SQL, XSS), IDS Suricata (27+ règles), fingerprint JA3, analyse d'entropie, corrélation d'attaques multi-étapes et vérification contre les threat feeds.
3
Décision — Le paquet reçoit un statut : PERMIS, ALERTE ou BLOQUÉ. Si bloqué, une règle est ajoutée au pare-feu Windows via netsh advfirewall.
4
Visualisation — L'état est envoyé chaque seconde au dashboard via WebSocket. Tu vois tout en temps réel : paquets, menaces, carte mondiale, statistiques.
Lancement rapide
python netguard.py
python netguard.py --no-block
python netguard.py --interface Wi-Fi
Mini Formation — Cybersecurite pour Debutants NOUVEAU
Bienvenue dans votre formation express !
Cette mini formation vous guide pas a pas pour comprendre et utiliser NetGuard Pro, meme si vous n'avez aucune experience en cybersecurite. Suivez les modules dans l'ordre.
Module 1 C'est quoi la cybersecurite ?
▶
La cybersecurite c'est proteger vos ordinateurs, vos donnees et votre reseau contre les pirates informatiques.

Pourquoi c'est important ?
• 60% des PME ferment dans les 6 mois apres une cyberattaque
• Les pirates scannent internet 24h/24 a la recherche de cibles faciles
• Vos donnees personnelles (mots de passe, cartes bancaires) valent de l'argent

Bonne nouvelle : NetGuard Pro fait tout ca automatiquement pour vous !
Module 2 Premier lancement — Que faire ?
▶
Etape 1 : Lancez NetGuard Pro (double-clic sur l'icone du bureau)
Etape 2 : Le dashboard s'ouvre automatiquement — vous voyez le trafic reseau en direct
Etape 3 : Le point vert en haut a droite = tout fonctionne !
Etape 4 : Laissez tourner — NetGuard Pro surveille en arriere-plan

Astuce : Cliquez sur "Lancer Suite NetGuard" sur votre bureau pour demarrer TOUS les agents de securite d'un seul coup.
Module 3 Comprendre le Dashboard
▶
En haut : Les metriques principales (paquets analyses, menaces detectees, IPs bloquees)
A gauche : Le menu de navigation — chaque section a un role precis

Les couleurs importantes :
● Vert = Tout va bien, trafic normal

Module 5 Que faire si une menace est detectee ?
▶
Pas de panique ! NetGuard Pro bloque automatiquement les menaces. Voici quoi faire :

1. Allez dans l'onglet Menaces pour voir les details
2. Les IPs dangereuses sont deja bloquees (onglet IPs bloquees)
3. Si vous recevez beaucoup d'alertes du meme pays, activez le Geoblocage
4. Pour etre notifie, configurez Discord ou Telegram dans les parametres

Urgence : Si vous pensez etre pirate, deconnectez votre cable reseau/Wi-Fi et contactez contact@netguardpro.com
Module 6 Les Notifications Internes
▶
Chaque agent de la suite NetGuard Pro vous envoie des notifications en temps reel :

• Alerte Rouge : Menace critique bloquee (scan de ports, brute force, malware)
• Alerte Orange : Activite suspecte a surveiller (trafic inhabituel, DNS suspect)
• Notification Verte : Action reussie (IP bloquee, scan termine, VPN connecte)
• Info Bleue : Mise a jour de statut (agent demarre, baseline mise a jour)

Ou voir les notifications ?
→ Dans le Dashboard : flux en direct
→ Dans Sentinel OS : vue centralisee de tous les agents
→ Sur Discord/Telegram : si configure dans les parametres

Agents & Architecture de la Suite
Mode Agent — Comment ca fonctionne
NetGuard Pro fonctionne en mode agent autonome. Chaque programme de la suite tourne independamment et communique via WebSocket avec le centre de commande Sentinel OS.

Les agents s'echangent des informations via un bus de messages pub/sub (publish/subscribe). Quand un agent detecte une menace, tous les autres sont alertes instantanement.
Sentinel OS — Orchestrateur
Port 8900
Centre de commande qui connecte et controle tous les agents. Correlation de menaces multi-sources, moteur de playbooks automatises, alertes Discord/Telegram, chat IA contextuel.
NetGuard Pro — IDS/Firewall
Port 8765
Moteur principal de surveillance reseau. Capture et analyse chaque paquet, detecte les attaques (scan de ports, brute force, SYN flood, DPI), bloque les IP malveillantes via le pare-feu OS.
VPN Guard — WireGuard VPN
Port 8820
VPN WireGuard avec Kill Switch, protection DNS, split tunneling, reconnexion Wi-Fi automatique, multi-profils. Chiffre tout votre trafic internet.
CleanGuard — Antimalware
Port 8810
Scanner antimalware + nettoyeur systeme. Signatures YARA, hashes SHA256, detection EICAR, scan de processus, nettoyage navigateurs, registre Windows, quarantaine.
MailShield — Securite Email
Port 8801
Client email securise avec detection phishing multi-criteres, filtrage spam, analyse de pieces jointes, support OAuth2 Microsoft, blacklist/whitelist.
StrikeBack — Red Team
Port 8850
Reponse active contre les attaquants. Tarpit TCP (ralentit les pirates), deploiement de decoys, tracking d'IP, scoring de danger, isolation automatique.
HoneyPot — Pieges
Port 8830
Deploie des fichiers pieges (faux mots de passe, fausses cles SSH) et des faux services (SSH, HTTP, FTP). Detecte toute tentative d'intrusion.
FIM — Integrite Fichiers
Port 8840
Surveille les fichiers systeme critiques. Cree une baseline de hashes et detecte toute modification, suppression ou ajout suspect.
Recorder — Enregistreur
Port 8860
Enregistre les sessions reseau, surveille les processus, les changements de fichiers et le trafic. Genere des rapports d'incidents detailles.
