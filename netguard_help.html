Comment fonctionne NetGuard Pro
🔄
Architecture en 3 couches
NetGuard Pro fonctionne avec un backend Python qui capture les paquets réseau, un moteur de détection qui analyse chaque paquet en temps réel, et un tableau de bord web connecté via WebSocket pour tout visualiser.
1
Capture — Scapy intercepte tous les paquets réseau qui transitent par ton interface (Wi-Fi ou Ethernet) grâce au driver Npcap sur Windows.
2
Analyse — Chaque paquet est analysé : IP source, destination, port, protocole, flags TCP. Le moteur vérifie contre les règles actives, les listes noires et les seuils de détection.
3
Décision — Le paquet reçoit un statut : PERMIS, ALERTE ou BLOQUÉ. Si bloqué, une règle est ajoutée au pare-feu Windows via netsh advfirewall.
4
Visualisation — L'état est envoyé chaque seconde au dashboard via WebSocket. Tu vois tout en temps réel : paquets, menaces, carte mondiale, statistiques.
Lancement rapide
python netguard.py --demo
python netguard.py --no-block
python netguard.py --interface Wi-Fi
