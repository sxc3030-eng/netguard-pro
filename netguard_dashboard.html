À propos

NetGuard Pro — Moteur Python + tableau de bord WebSocket

Backend: python netguard.py --interface auto

Sans blocage: python netguard.py --no-block

Démonstration: python netguard.py --demo

Requires: scapy, websockets — pip install scapy websockets

Linux: sudo requis pour la capture. Windows: Npcap driver requis.
